# Spirit of Adventures (React v.18)

Aggregator for 'craft' travel routes

'Spirit of Adventures' makes your life more exciting every day.

## Features:
* Create your own travel plan and share it with community
* Use other user's routes to get unique travel experience
* Find route perfectly fitting to your requirements (longtitude, transport, type, activities, etc.)

### Stack:

* **Frontend:**
  * React v.18
  * TypeScript
  * Redux
  * Redux-Thunk
  * Material UI
* **Backend:**
  * Node.JS
  * Express
  * Mongoose
* **Database:**
  * MongoDB
